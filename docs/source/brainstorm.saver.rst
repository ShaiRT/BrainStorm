brainstorm.saver package
========================

A package for a saver that reads data from a message queue and save to a database.
This package uses the drivers from the ``brainstrom.database_drivers`` and ``brainstorm.mq_drivers`` packages.

See `Saver <https://brainstormproject.readthedocs.io/en/latest/README.html#saver>`_ documentation for CLI usage of the package.
